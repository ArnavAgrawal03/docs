## Usage

<Tabs>
  <Tab title="Sync">
    ```python
    from databridge.sync import DataBridge
    
    # Without authentication
    db = DataBridge()
    
    # With authentication
    db = DataBridge("databridge://owner_id:token@api.databridge.ai")
    ```
  </Tab>
  <Tab title="Async">
    ```python
    from databridge.async_ import AsyncDataBridge
    
    # Without authentication
    async with AsyncDataBridge() as db:
        doc = await db.ingest_text("Sample content")
    
    # With authentication
    async with AsyncDataBridge("databridge://owner_id:token@api.databridge.ai") as db:
        doc = await db.ingest_text("Sample content")
    ```
  </Tab>
</Tabs>

## Constructor

Both clients share the same constructor parameters:

<Tabs>
  <Tab title="Sync">
    ```python
    DataBridge(uri: Optional[str] = None, timeout: int = 30, is_local: bool = False)
    ```
  </Tab>
  <Tab title="Async">
    ```python
    AsyncDataBridge(uri: Optional[str] = None, timeout: int = 30, is_local: bool = False)
    ```
  </Tab>
</Tabs>

### Parameters

- `uri` (str, optional): DataBridge URI in format "databridge://&lt;owner_id&gt;:&lt;token&gt;@&lt;host&gt;". If not provided, connects to http://localhost:8000 without authentication.
- `timeout` (int, optional): Request timeout in seconds. Defaults to 30.
- `is_local` (bool, optional): Whether connecting to local development server. Defaults to False.

## Context Manager

Using the DataBridge client as a context manager ensures that resources are properly closed when the context exits.

<Tabs>
  <Tab title="Sync">
    ```python
    with DataBridge() as db:
        doc = db.ingest_text("Sample content")
    ```
  </Tab>
  <Tab title="Async">
    ```python
    async with AsyncDataBridge() as db:
        doc = await db.ingest_text("Sample content")
    ```
  </Tab>
</Tabs>
