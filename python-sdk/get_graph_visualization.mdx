## Parameters

- `name` (str): Graph name.
- `folder_name` (str | List[str], optional): Folder scope filter.
- `end_user_id` (str, optional): End-user scope filter.

## Returns

- `Dict[str, Any]`: JSON with `nodes` and `links` arrays ready for d3-force or similar libraries.

## Example

```python
viz = db.get_graph_visualization("research_graph")
print(len(viz["nodes"]), "nodes", len(viz["links"]), "edges")
```
