---
title: 'How do I integrate Morphik into my application?'
description: 'Overview of using the API and SDKs in production'
---

Morphik offers a REST API and a Python SDK. For simple workflows you can call the API directly:

```bash
curl -X POST https://api.morphik.ai/ingest -H 'Authorization: Bearer YOUR_KEY' -d '{"text":"Hello"}'
```

For more complex pipelines, use the SDK:

```python
from morphik import Morphik
client = Morphik(api_key="YOUR_KEY")
client.ingest_text("docs/")
```

See [API reference](/api-reference/ingest-text) and [Python SDK](/python-sdk/morphik) for all available endpoints.
