---
title: 'What is DataBridge?'
description: 'An overview of Retrieval Augmented Generation and DataBridge'
---

DataBridge is a multi-modal RAG system that allows you to store, search, and retrieve data from various sources.

## Features

- Store and retrieve data from various sources
- Search and retrieve data using natural language queries
- Retrieve data using vector search
- Retrieve data using similarity search
- Retrieve data using semantic search
