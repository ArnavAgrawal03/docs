---
title: 'What is DataBridge?'
description: 'An overview of Retrieval Augmented Generation and DataBridge'
---

DataBridge is a multi-modal RAG system that allows you to store, search, and retrieve data from various sources. It is built on top of the [LangChain](https://www.langchain.com/) framework and the [Chroma](https://www.chromadb.org/) vector store.
